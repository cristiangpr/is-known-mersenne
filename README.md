# Is Known Mersenne

Quickly determines if a number is a known mersenne prime. Returns boolean value


### Prerequisites

Node 10.4 or later


### Installation





```
npm install is-known-mersenne
```

Usage
```
isKnownMersenne(number)
```



## Running the tests



```
npm test
```
